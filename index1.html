head section


 in html head element is a container for the following elements: title, style, meta, link, script, and base 
title- to define the title of the document
style- to define style information for a single html page color,font,size etc
link- to define relationship between current documnet and external source or element
meta- to specify document details like author name, page description, characters set, keywords, viewport(user's visible area of a web page depends on different device sizes)settings etc commented below

script- to define client side Javascripts
base- to specify a base url for all the urls used in the page(like google.com as base for page using google images links,google map links,etc). there can only be one base element in a document







HTML layout


html consists of certain semantics that define different parts of a webpage:
header - defines header for a document or section of the page
nav - defines all navigation links
section - defines a section inside a document
article - defines an independent selfcontained content
aside - defines content aside from content(sidebar(having other breaking news etc))
footer - defines footer of a document or a section
details - defines additional details the user can open and close on demand 
summary - defines a heading for the element

the different types of layout techniques are 
CSS Framework - most commonly used. W3.CSS or bootstrap

CSS Float -it is easy to learn and work on. one must learn float and clear properties to implement it. Float&Clear

CSS Flexbox - it ensures that the elements behave predictable when the pagelayout must accomodate different screens and displays. FlexBox

CSS Grid - eliminates use of floats and positioning and isa grid-based layout system. Grid
